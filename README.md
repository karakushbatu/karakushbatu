### 🚀 About me

I'm a computer engineer who likes building things that are *hard to get right* — local-first AI, real-time avionics software, and the occasional 3D web experiment (like the portfolio above).

- 🧠 **AI systems** — on-device LLM assistants, RAG pipelines and an assistive-vision system for the blind.
- 🛰️ **Embedded &amp; real-time** — avionics software with ARINC 653, DO-178C and FreeRTOS (TUSAŞ SKY).
- 🔬 **Research** — TÜBİTAK STAR fellow; building the **pAGEs** food-toxicology ML platform (Project 224Z036).
- 🌍 **Languages** — Turkish (native) · English (C1) · German (A2)
- ⚡ **Off-screen** — kickboxing, guitar, chess, game design, coffee and philosophy.

> *"As passionate about game mechanics as I am about low-level systems."*
